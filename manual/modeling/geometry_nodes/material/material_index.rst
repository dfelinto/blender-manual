*******************
Material Index Node
*******************

.. figure:: /images/modeling_geometry-nodes_material_material-index_node.png
   :align: right

   Material Index Node

The *Material Index* node outputs the material index for a geometry.

The node to set this data is the :doc:`Set Material Index </modeling/geometry_nodes/material/set_material_index>` node.

Inputs
======

This node has no inputs.


Properties
==========

This node has no properties.


Outputs
=======

Material Index
   Standard integer value
