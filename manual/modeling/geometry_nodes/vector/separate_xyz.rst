Separate XYZ Node
*****************

.. figure:: /images/modeling_geometry-nodes_vector_separate-xyz_node.png
   :align: right

   Separate XYZ Node.

The *Separate XYZ Node* splits a vector into its individual components.


Input
=====

Vector
   Standard vector input.


Properties
==========

This node has no properties.


Outputs
=======

- X
- Y
- Z
