*****************
Separate RGB Node
*****************

.. figure:: /images/node-types_ShaderNodeSeparateRGB.webp
   :align: right
   :alt: Separate RGB Node.

The *Separate RGB Node* splits an image into its composite color channels.


Inputs
======

Image
   Standard color input.


Properties
==========

This node has no properties.


Output
======

- R (Red)
- G (Green)
- B (Blue)
