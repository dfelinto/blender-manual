***********
Combine RGB
***********

.. figure:: /images/modeling_geometry-nodes_color_combine-rgb_panel.png
   :align: right

   Combine RGB Node.

The *Combine RGB Node* combines an image from its composite color channels.


Inputs
======

- R (Red)
- G (Green)
- B (Blue)


Properties
==========

This node has no properties.


Output
======

Image
   Standard image output.
