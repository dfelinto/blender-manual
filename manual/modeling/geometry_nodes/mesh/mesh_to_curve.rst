******************
Mesh to Curve Node
******************

.. figure:: /images/modeling_geometry-nodes_mesh_mesh-to-curve_node.png
   :align: right

   Mesh to Curve node.

The *Mesh to Curve* node generates a curve from a mesh.


Inputs
======

Mesh
   Standard mesh input.

Selection
   The meshes that will be converted to curves.


Properties
==========

This node has no properties.


Outputs
=======

Curve
   Generated curve.
