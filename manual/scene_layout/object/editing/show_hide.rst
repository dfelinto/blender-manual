*********
Show/Hide
*********

.. reference::

   :Mode:      All Modes
   :Menu:      :menuselection:`Object --> Show/Hide`

Show Hidden Objects :kbd:`Alt-H`
   Reveals all hidden objects.
Hide Selected :kbd:`H`
   Hides all selected objects.
Hide Unselected :kbd:`Shift-H`
   Hides all unselected objects of the scene.
