
*************
Distance Node
*************

.. figure:: /images/editors_texture-node_types_converter_distance_node.png
   :align: right

   Distance node.

Computes the distance between two 3D coordinates.


Inputs
======

Coordinate 1
   First coordinate point.
Coordinate 2
   Second coordinate point.


Properties
==========

This node has no properties.


Outputs
=======

Value
   Calculated distance.
